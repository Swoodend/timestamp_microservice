Timestamp Microservie

This is a timestamp microservice completed for the Free Code Camp backend chalenges.


Example usage:

https://localhost:8080/December%2020,%202015

https://localhost:8080/1450137600

https://localhost:8080/December 20, 2015

https://localhost:8080/dec 20 2015

https://localhost:8080/dec 20, 2015


Example output:

{ "unix" : "1450137600", "natural" : "December 15, 2015" }
